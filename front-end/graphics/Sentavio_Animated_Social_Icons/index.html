Embedding SVG animation

Animated SVG icons can be used/embeded into your website/blog.

Variant 1. IFRAME (recommended)

  1. Choose your favourite icon above
  2. Click on SVG under icon, save file
  3. Upload icon SVG file to your webserver or place locally in desired folder
  4. Use IFRAME code from examples below

Example 1. Fixed width / height

<iframe width=YOURWIDTH height=YOURHEIGHT src="PATHTOYOURICON/ICONNAME.svg" frameborder=0 style="border:none;"></iframe>

Example 2. Bootstrap responsive

<div class="embed-responsive embed-responsive-4by3">
   <iframe class="embed-responsive-item" src="PATHTOYOURICON/ICONNAME.svg"></iframe>
</div>

Variant 2. IMG+JS

  1. Upload svglib, unzip and upload folder to your webserver
  2. Paste <script src="YOUR_SERVER_SVGLIB_FOLDER_URL/svglib.js"></script> anywhere on page with animation
    (for CMS like Wordpress, please include in footer).
  3. <img> tag requirements
    <img id="svg__holder" width="DESIRED_WIDTH" width="DESIRED_HEIGHT" src="YOUR_SERVER_SVGLIB_FOLDER_URL/ICON_FILENAME.svg">
    Warning Please, make sure that id="svg__holder" is present, elsewhere animation will not work

Embedding GIF animation

  1. Choose your favourite icon above
  2. Click on white grey black under icon depending on resulting background, save file
  3. For social media, forums, blogs etc. just upload and use the file
  4. For manual embedding via <img> tag
    <img width="DESIRED_WIDTH" width="DESIRED_HEIGHT" src="ICON_FILENAME.gif">

Notice for WordPress  You could not upload svg files into Media gallery. Because of security restrictions by default WordPress omits svg in media gallery.

Facebook profile video avatar

  1. Choose your favourite icon above
  2. Click on MP4 and save icon video file
  3. Send this file to your iPhone / iPad (via email / iTunes etc); the idea is to save the file in gallery on your device
  4. Open facebook application on iPhone / iPad, edit your profile picture and you will be able to choose the video from gallery
  5. New avatar will be also viewable on desktop facebook after changing on device
